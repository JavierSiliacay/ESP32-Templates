# ESP32 Templates

Welcome to my ESP32 templates repository! 

This repository contains a collection of ESP32 and ESP32-CAM examples, templates, and projects. It was originally sourced from various open-source snippets and examples, and organized here for templates and reference for future projects.

## Contents
- **ESP32_Examples/**: Contains all the sketches and projects. Including examples for WebServer, AI integration, Sensors, and more.

## Development Environment
- Arduino IDE 
- Arduino core for the ESP32
